# dawe2026cicd2
Pràctica RA6 CICD intro

**Kilian Cuadrado Barbero**
> [!NOTE] 
> La pàgina de index.html està dividida en capçalera, un bloc per afegir tasques (formulari amb text + etiqueta), dues llistes separades per a pendents i completades, i un peu de pàgina.
> El fitxer style.css està organitzat per blocs molt clars (variables a :root, estils base, capçalera, contingut principal, formulari, llistes, peu i responsive), de manera que és fàcil personalitzar colors, tipografia i espaiats des de les variables.
> El JavaScript de app.js gestiona lògica completa de l’app: carrega i desa tasques a localStorage, renderitza les llistes amb comptadors, permet marcar/desmarcar tasques amb clic i afegir-ne de noves des del formulari.
